Farm Management in Odoo

This app allows you to manage a farm and agriculture business as per the below-listed features and screenshots.


Main Features:
  • Agriculture / Crop Requests: Allow users to create crop requests for different crops during periods or seasons.
  • Agriculture / Crops / Crops: Allow you to configure crops that your business is dealing with and also allow you to set up things like crop materials, labor, overheads, diseases, processes, and descriptions for that crop as shown.
  • Agriculture / Crops / Process Template : Configuration of processes of crops.
  • Agriculture / Incidents: Different incidents of crops which may occur while doing crop in the farm.
  • Agriculture / Diseases Cures: Diseases cures of crops.
  • Agriculture / Projects: The system will create projects of crops from crop requests.
  • Agriculture / Tasks: The system will create tasks for crop projects so you can manage crop jobs on the farm using odoo project task management of odoo.
  • Agriculture / Configuration / Farmers: Create and manage by your farmers.
  • Agriculture / Configuration / Farm Locations : Farm locations of different farmers.
  • Agriculture / Configuration / Animals: Animals setup which is used for the crops process.
  • Agriculture / Configuration / Fleets : Setup of fleets which are used for crops.
  • Agriculture / Configuration / Stages : Different tasks stages of crops.
  • Agriculture / Crop Requests : Print PDF - Crop Requests Report.
  • Agriculture / Crops : Print PDF crops report.
  • For more details please check the below screenshots and watch the video.
